HTML ย่อมาจาก Hypertext Markup Language ซึ่งเป็นภาษาคอมพิวเตอร์ที่ไว้ใช้เเสดงผลในเว็บบราวเซอร์

HTML Elements

ใน html จะประกอบไปด้วย elements หลายๆ ตัว โดยที่จะต้องมี Tag เปิดและปิด
 - Tag เปิดจะมีชื่อ elements ในเครื่องหมาย <...>
 - Tag ปิดจะเหมือน Tag เปิดแต่จะมี forward slash เป็นตัวบอกจุดสิ้นสุด ดังนี้ </...>

ตัวอย่าง elements
  - <h1> ใส่ heading </h1> โดยจะมี h1(หัวข้อใหญ่สุด) จนไปถึง h6(หัวข้อเล็กสุด)
  - <p> paragraph </p> โดยจะทำการเพิ่มบรรทัดว่างก่อนและหลังเนื้อหาที่เราพิมพ์
  - <br> เป็นคำสั่งขึ้นบรรทัดใหม่ โดยจะวางไว้หลังสุดของประโยค ซึ่งจะไม่มี Tag ปิด

Nested HTML Elements

 ภายใน Elements สามารถมี Elements อื่นๆ ซ้อนอยู่ได้ แต่ต้องระวังในการจัดลำดับ Tag เปิดและปิดให้ถูกต้อง
ตัวอย่าง

<!DOCTYPE html>
<html>
<body>

<h1> My First Heading<h1>
<p> My first paragraph<p>

</body>
</html>

  จากตัวอย่างข้างต้น จะมี Tag เปิด <html> และ tag ปิด </html> เสมอ ซึ่งเป็นการระบุว่าเป็น HTML ในส่วนของ <body> จะเป็นการระบุของเนื้อหาที่จะเเสดงผล โดยจะมี Tag เปิด <body> และ tag ปิด </body> เสมอ และ Elements ที่ระบุรูปแบบเเสดงผล จะมี <h1> เป็นการระบุหัวข้อ โดยจะมี Tag เปิด <h1> และ tag ปิด </h1> เสมอ และ <p> เป็นการระบุย่อหน้า โดยจะมี Tag เปิด <p> และ tag ปิด </p> เสมอ

Never Skip the End Tag

  ในบางครั้งที่มีการลืมปิด Tag แต่ก็ยังที่จะสามารถแสดงผลได้ถูกต้อง แต่ในทางที่ดีที่สุดควรระมัดระวังไม่ให้ลืม เพื่อไม่ให้เกิดข้อผิดพลาดได้
ตัวอย่าง

<!DOCTYPE html>
<html>
<body>

<h1> My First Heading
<p> My first paragraph

</body>
</html>

Empty HTML Elements

 คือ HTML Elements ที่ไม่มีเนื้อหา เช่น <br> ที่ไว้ใช้ในการเว้นบรรทัด ซึ่งจะไม่มี Tag ปิด
ตัวอย่าง

<p> This is a <br> paragraph with a line break.

HTML is Not Case Sensitive

 การใช้ Tag ตัวพิมพ์เล็กหรือตัวพิมพ์ใหญ่ จะไม่มีผลต่อการเเสดงผล ซึ่งส่วนมากจะนิยมใช้ตัวพิมพ์เล็ก

HTML Attributes

Attributes เป็นการระบุข้อมูลเพิ่มเติมของ Elements โดยจะระบุชื่อและค่า( name="value")ใน Tag เปิดเท่านั้น

The href Attribute

 ในการเชื่อมโยงข้อมูลใส่ลิงก์นั้น จะมีการระบุ Tag <a> โดยมี Attribute href เพื่อใส่ url ลิงก์ในนั้น
ตัวอย่าง

<a href="https://www.w3schools.com" > Visit W3Schools </a>

ผลลัพธ์

Visit W3Schools

The src Attribute

 ในการใส่รูปภาพนั้น จะมีการระบุ Tag <img> โดยมี Attribute src เพื่อทำการระบุไฟล์รูปภาพในนั้น หรือเราสามารถใส่ระบุเป็นลิงก์ของรูปภาพได้
ตัวอย่าง

<img src="doraemon2.jpg" >

ผลลัพธ์

The width and height Attributes

 ภายใน Tag <img> เราสามารถระบุขนาดของไฟล์รูปภาพได้ใน Attribute width, height
ตัวอย่าง

<img src="doraemon.jpg" width="200" height="200" >

ผลลัพธ์

The alt Attribute

 เป็นการแสดงข้อความในกรณีที่โหลดรูปภาพไม่ขึ้น
ตัวอย่าง

<img src="doraemon3.jpg" alt="Doraemon" >

ผลลัพธ์

The style Attribute

 เป็นการตกแต่งหน้าเว็บ เช่น สี ขนาด ตัวอักษร เป็นต้น โดยกำหนดที่ Attribute style
ตัวอย่าง

<style="color:red;">This is a red paragraph.</p>

ผลลัพธ์

This is a red paragraph.

The lang Attribute

 เป็นการระบุภาษาที่ใช้ในเว็บ โดยกำหนดที่ Attribute lang ไว้ใน Tag <html>
ตัวอย่าง

<!DOCTYPE html>
<html lang="en">
<body>
<h1> ...
</body>
</html>

The title Attribute

  เมื่อกำหนด Attribute title ไว้ใน Tag ค่าของ title ที่เราใส่ไว้ คำอธิบายเพิ่มเติมจะเเสดงเมื่อเราเอาเม้าส์ไปวางไว้บน Elements <p>
ตัวอย่าง

<p title="I'm a tooltip">This is a paragraph.</p>

HTML <blockquote> for Quotations

Internal CSS

    h1   {color: blue;}
    p    {color: red;}
    </style>

    </body>
    </html>

    </head>

CSS Border

    การกำหนดค่ากรอบ โดยการใช้ property border ในการกำหนด
    ตัวอย่าง

    p {
        border: 2px solid powderblue;
      }     



CSS Padding

    กำหนดขนาดช่องว่างระหว่างข้อความภายในกับกรอบ โดยการใช้ property padding ในการกำหนด
    ตัวอย่าง

    p {
        border: 2px solid powderblue;
        padding: 30px;
      }   



CSS Margin

    กำหนดระยะห่างภายนอกกรอบ โดยการใช้ property margin ในการกำหนด
    ตัวอย่าง

    p {
        border: 2px solid powderblue;
        margin: 50px;
      }



Link to External CSS

    การอ้างอิงไฟล์
    - ตัวอย่างใช้ URL ในการลิงก์ไปหา CSS เว็บไซต์อื่น เช่น <link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">
    - ตัวอย่างลิงก์ไปหา CSS ในเว็บไซต์ตัวเอง เช่น  <link rel="stylesheet" href="/html/styles.css" >
    - ตัวอย่างลิงก์ไปหา CSS ในเว็บไซต์ตัวเองที่อยู่ในโฟลเดอร์เดียวกัน เช่น  <link rel="stylesheet" href="styles.css">
