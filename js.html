JavaScript Where To

The <script> Tag 

        ใน HTML โค้ดของ JavaScript จะอยู่ระหว่าง Tag <script> และ </script> 
    ตัวอย่าง
<script>document.getElementById("demo").innerHTML = "My First JavaScript";</script>

JavaScript in <head> and <body>

        เราสามารถใส่สคริปต์ใน <head> หรือ <body>ก็ได้

JavaScript Output

Using innerHTML

    จะใช้ document.getElementById(id) โดย id จะเป็นตัวกำหนดตัว HTML element 
และproperty innerHTML จะกำหนด HTML content

<!DOCTYPE html>
    <html>
    <body>
    
    <h1>My First Web Page</h1>
    <p>My First Paragraph</p>
    <p id="demo"></p>
    
    <script>
    document.getElementById("demo").innerHTML = 5 + 6;
    </script>
    
    </body>
    </html>

Using document.write()

    จะใช้ document.write() ซึ่งในเมธอดนี้จะมีข้อมูลอยู่ เเละเมื่อ HTML document ถูกเรียกใช้งาน จะทำให้ HTML ทั้งหมดถูกลบ เหลือแต่ข้อมูลที่อยู่ใน document.write() เท่านั้น 
ตัวอย่าง

<!DOCTYPE html>
    <html>
    <body>
    
    <h1>My First Web Page</h1>
    <p>My First Paragraph</p>
    
    <script>
    document.write(5 + 6);
    </script>
    
    </body>
    </html>

Using window.alert()

    เป็นกล่องเเจ้งเตือน pop up
ตัวอย่าง

<!DOCTYPE html>
    <html>
    <body>
    
    <h1>My First Web Page</h1>
    <p>My First Paragraph</p>
    
    <script>
    window.alert(5 + 6);
    </script>
    
    </body>
    </html>

Using console.log()

    เป็นการแสดงข้อความออกมาบน console ใน browser
ตัวอย่าง

<!DOCTYPE html>
    <html>
    <body>
    
    <script>

    </script>
    
    </body>
    </html>

JavaScript Print

    เป็นการแสดงผลข้อมูลในหน้าจอที่เรากำลังใช้งาน
ตัวอย่าง

<!DOCTYPE html>
    <html>
    <body>
    
    <h2>The window.print() Method </h2>
    <p>Click the button to print the current page.</p>
    <button onclick="window.print()">Print this page</button>

    </body>
    </html>

JavaScript Statements

Semicolons ;

    ในการเพิ่ม Semicolons หลังคำสั่งในแต่ละคำสั่ง จะทำให้ใช้หลายคำสั่งในหนึ่งรรทัดได้

JavaScript White Space

    JavaScript ไม่สนใจตัว spaces หรือเว้นระยะ ว่ามากเพียงใด ซึ่งจุดประสงค์ของการใช้ white space เพื่อให้โค้ดอ่านได้ง่ายขึ้น

JavaScript Line Length and Line Breaks

    เพื่อให้โค้ดที่เขียนนั้นอ่านได้ง่าย ส่วนใหญ่จะหลีกเลี่ยงการเขียนโค้ดที่มีความยาวเกินกว่า 80 ตัวอักษรภายในบรรทัดเดียวกัน

JavaScript Code Blocks

    JavaScript statements สามารถจัดกลุ่มรวมกันในบล็อค โดยจะใช้เครื่องหมายปีกกา {} (Curly Brackets)

JavaScript Syntax

JavaScript Literals

1. Numbers สามารถเขียนได้ทั้งจำนวนเต็ม และทศนิยม 
2. Strings สามารถใช้ '(single quote) หรือ  " (double quote) ก็ได้


JavaScript Variables

    JavaScript จะใช้แค่คีย์เวิร์ด var ตัวเดียวเพื่อประกาศตัวแปร เช่น
    var x;
    x = 6;

JavaScript Operators

    ใช้เครื่องหมาย + - * / ทางคณิตศาสตร์มนการคำนวณค่า และใช้ = เพื่อกำหนดค่าให้ตัวแปร

JavaScript Comments

    ในการคอมเมนต์ อธิบายโค้ดเพิ่มเติมเพื่อให้เข้าใจ โดยจะไม่มีผลต่อการแสดงผลบนเว็บ โดยจะมีลักษณะการพิมพ์ดังนี้
    // จะคอมเม้นต์ 1 บรรทัด เเละสามารถคอมเม้นหลายๆ บรรทัด โดยใช้ /* ไว้ด้านหน้าสุด เเละ */ หลังสุดของข้อความ

JavaScript Comments

- คอมเม้นต์บรรทัดเดียว จะใช้ // ข้างหน้าข้อความ เช่น //ข้อความที่จะคอมเม้นต์ - คอมเม้นต์หลายบรรทัด จะใช้/* ไว้ด้านหน้าสุด เเละ */ หลังสุดของข้อความ เช่น /*ข้อความคอมเม้นต์บรรทัดที่ 1 บรรทัดที่ 2 */
