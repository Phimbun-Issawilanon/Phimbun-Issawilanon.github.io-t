JavaScript Comments

    ในการคอมเมนต์ อธิบายโค้ดเพิ่มเติมเพื่อให้เข้าใจ โดยจะไม่มีผลต่อการแสดงผลบนเว็บ โดยจะมีลักษณะการพิมพ์ดังนี้

JavaScript Comments

    - คอมเม้นต์บรรทัดเดียว จะใช้ // ข้างหน้าข้อความ เช่น //ข้อความที่จะคอมเม้นต์
    - คอมเม้นต์หลายบรรทัด จะใช้/* ไว้ด้านหน้าสุด เเละ */ หลังสุดของข้อความ เช่น 
      /*ข้อความคอมเม้นต์บรรทัดที่ 1
      บรรทัดที่ 2 */

    หรือขีดล่าง (_) เช่น

JavaScript Data Types

    Javascript เป็น Dynamic Typed คือประกาศตัวแปรโดยไม่ต้องบอก Type ว่าเป็น Int Float หรือ String 

JavaScript Strings

JavaScript Numbers

ตัวเขที่มีค่ามากและน้อยมากอาจเขียนได้ในรูปแบบดังนี้เพื่อให้ดูง่ายขึ้น
    var y = 123e5;      จะได้ผลลัพธ์ดังนี้ --> 12300000
    var z = 123e-5;     จะได้ผลลัพธ์ดังนี้ --> 0.00123

JavaScript Booleans

    boolean ใช้เก็บข้อมูล 2 ค่าคือ true และ false 

JavaScript Arrays

    ชุดของข้อมูล จะใช้โดยใช้ brackets [] และคั่นแต่ละสมาชิกด้วย comma(,) และในการเข้าถึงของอาร์เรย์ค่าเริ่มต้นของ Index จะมีค่าเท่ากับ 0 เสมอ
ตัวอย่าง: var cars = ["Saab", "Volvo", "BMW"];

JavaScript Objects

    จะใช้ปีกกา {} โดยจะเก็บข้อมูลเป็น Key และ Value(เรียกว่า Property) และถ้ามีหลาย Property จะคั่นแต่ละค่าด้วยเครื่องหมาย comma(,)
ัตัวอย่าง: var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

The typeof Operator

    ตรวจสอบประเภทข้อมูลในตัวแปร ซึ่งจะส่งค่ากลับมาเป็น String ตัวอย่างเช่น
  - typeof "John"   จะส่งค่ากลับมาเป็น  "string" (สำหรับข้อมูลที่เป็นข้อความ)
  - typeof 314      จะส่งค่ากลับมาเป็น  "number" (สำหรับข้อมูลที่เป็นตัวเลข)

Undefined

    ตัวแปรที่ประกาศเอาไว้แต่ไม่ได้มีการกำหนดค่าใดๆ เช่น
    var y; 

Empty Values

 เป็นตัวแปรสตริงว่าง ตัวอย่างเช่น var car = "";   

Null

    ตัวแปรที่ไม่มีค่าใดๆเลย ไม่เท่ากับ 0 ไม่เท่ากับสตริงว่าง และไม่สามารถเอาไปใช้ในการคำนวณได้ แต่สามารถนำไปเชตค่าได้

JavaScript Functions

    เป็นชุดคำสั่งที่ใช้ในการทำงานอย่างใดอย่างหนึ่ง ซึ่งจะทำงานเมื่อถูกเรียกใช้งาน

JavaScript Function Syntax

    จะมีลักษณะการเขียนดังนี้ 
    function  ชื่อฟังกชั่น (อาร์กิวเมนต์)
    { คำสั่งต่างๆ  }
ซึ่งในการตั้งชื่อฟังก์ชันนั้นจะมีหลักการคล้ายกับการตั้งชื่อตัวแปร ส่วนใหญ่จะนิยมใช้ตัวพิมพ์เล็ก 

Function Return

  เป็นฟังก์ชันที่ส่งค่ากลับ โดยการส่งค่ากลับจากฟังก์ชันจะใช้คำสั่ง return
ตัวอย่าง

var x = myFunction(4, 3); 

function myFunction(a, b) {
  return a * b;             
}

ผลลัพธ์ x จะมีค่าเป็น 12

JavaScript Events

Event คือ เหตุการณ์ที่เกิดขึ้นขณะผู้ใช้กระทำกับวัตถุหรือสิ่งต่าง ๆ ที่เกิดขึ้นบนจอภาพ เช่น การคลิกปุ่ม การวางเมาส์หรือการกดแป้นพิมพ์ เป็นต้น ตัวอย่าง onClick เป็นเหตุการณ์ที่ผู้ใช้ใช้เมาส์คลิกที่ปุ่ม <button onclick="document.getElementById('demo').innerHTML = Date()">The time is?</button>

Event ความหมาย
onchange เหตุการณ์ เมื่อเลือกตัวเลือก เช่นจาก dropdown
onclick เหตุการณ์ เมื่อเราคลิก
onmouseover เหตุการณ์ เมื่อเรานำเมาส์ไปวาง
onmouseout เหตุการณ์ ที่เมื่อเรานำเมาส์ออก
onkeydown เหตุการณ์เมื่อกดปุ่มบนคีย์บอร์ด
onload เหตุการณ์เมื่อมีการเปิดหน้าเว็บ

JavaScript String Methods

    - lenght ใช้หาความยาวของสตริง
    - indexOf() ใช้ค้นหาข้อตวามในสตริง เมื่อเจอครั้งเเรกจะรีเทิร์นค่าออกมาเ็นตำแหน่ง index
      ตัวอย่าง: var str = "Please locate where 'locate' occurs!";
              var pos = str.indexOf("locate");
      ผลลัพธ์: 7
    - lastIndexOf() ใช้ค้นหาข้อความที่เจอลำดับสุดท้ายและจะรีเทิร์นค่าออกมาเ็นตำแหน่ง index
      ตัวอย่าง: var str = "Please locate where 'locate' occurs!";
              var pos = str.lastIndexOf("locate");
      ผลลัพธ์: 21
    - trim() จะมีไว้ในการตัดช่องว่าง ที่วางไว้ด้านหน้าและหลังข้อความทิ้งไป
      ตัวอย่าง: console.log(" Phimbun Issawilanon ".trim() );
      ผลลัพธ์: "Phimbun Issawilanon"

JavaScript Numbers

    ตัวเลขจะมีทั้งจำนวนเต็มเเละทศนิยม 
- ตัวเขที่มีค่ามากและน้อยมากอาจเขียนได้ในรูปเลขยกกำลังฐานสิบโดยใช้ e แทนเลข 10 ยกกำลังดังนี้
     var y = 123e5;      จะได้ผลลัพธ์ดังนี้ --> 12300000
     var z = 123e-5;     จะได้ผลลัพธ์ดังนี้ --> 0.00123
- NaN หรือ Not-A-Number คือค่าที่ไม่สามารถคิดออกมาเป็นตัวเลขได้ ใน JavaScript จะมีตัวเเปร NaN เก็บค่าดังกล่าว
- ค่าอนันต์หรือ infinity คือค่าที่ไม่มีขอบเขต ไม่มีสิ้นสุด ใน JavaScript จะมีตัวเเปร infinity เก็บค่าดังกล่าว
